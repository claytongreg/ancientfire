## Local preview

Open any HTML file in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy to Vercel

This repo is a pure static site — no build step needed.

1. Push to GitHub
2. In Vercel, **New Project** → import this repo
3. Framework preset: **Other**
4. Build command: *leave empty*
5. Output directory: *leave empty* (or `.`)
6. Deploy

`vercel.json` enables clean URLs (`/about` instead of `/about.html`).
